## Part 1. 서비스 이용약관

### 제1조 (목적)

본 약관은 haru (이하 "회사") 가 제공하는 보이스 채팅 기반 크로스언어 매칭 서비스
(이하 "서비스") 의 이용 조건 및 절차, 회사와 회원의 권리·의무 및 책임 사항을
규정함을 목적으로 합니다.

### 제2조 (정의)

본 약관에서 사용하는 용어의 정의는 다음과 같습니다.

1. "회원" — 본 약관에 동의하고 서비스 이용 계약을 체결한 자.
2. "보이스 클론" — 회원이 등록한 음성 샘플로부터 ElevenLabs 의 음성 합성 모델이
   생성한 회원 고유의 음성 ID 및 합성 결과물.
3. "메시지 음성" — 회원이 채팅에서 전송한 텍스트가 발신자의 보이스 클론으로
   합성·번역되어 수신자에게 재생되는 음성 파일.
4. "보이스 한마디" — 회원이 프로필에 등록하는 짧은 자기소개 텍스트와 이를
   보이스 클론으로 합성한 음성. 매칭 탐색 단계에서 다른 회원에게 노출됩니다.
5. "콘텐츠" — 회원이 서비스 내에서 작성·전송·게시하는 모든 텍스트, 음성,
   사진을 통칭합니다.

### 제3조 (약관의 게시와 개정)

1. 회사는 본 약관의 내용을 회원이 쉽게 알 수 있도록 서비스 초기 화면에
   게시합니다.
2. 회사는 관련 법령에 위배되지 않는 범위에서 본 약관을 개정할 수 있으며,
   개정 시 적용 일자 및 개정 사유를 명시하여 시행일로부터 최소 7 일 전 (회원에게
   불리한 개정의 경우 최소 30 일 전) 부터 시행일까지 공지합니다.

### 제4조 (회원가입 및 본인 확인)

1. 서비스 이용자는 만 18 세 이상이어야 가입할 수 있습니다.
2. 회원은 가입 시 회사가 요구하는 정보 (이메일, 생년월일, 성별, 국적, 사용 언어,
   프로필 사진, 보이스 클론용 음성 샘플) 를 제공해야 합니다.
3. 회사는 신고 누적 또는 미성년자 의심 사유로 가입 거부 및 계정 정지 조치를
   취할 수 있습니다.

### 제5조 (서비스의 제공)

1. 회사는 회원에게 다음 서비스를 제공합니다.
   - 매칭 탐색 및 좋아요·매치
   - 매칭 상대와의 채팅 (텍스트, 자동 번역, 발신자 보이스 클론 음성 합성)
   - 보이스 한마디 등록 및 자동 번역·합성
2. 서비스의 일부는 외부 서비스 (ElevenLabs, OpenAI, Google Vertex AI, Supabase)
   를 사용합니다. 자세한 처리 위탁 내역은 개인정보처리방침 제3조 참조.

### 제6조 (회원의 의무)

회원은 다음 행위를 해서는 안 됩니다.

1. 타인을 사칭하거나 허위 정보를 등록하는 행위.
2. 미성년자가 가입하거나 미성년자임을 알고도 접근을 시도하는 행위.
3. 성적, 마약 관련, 자해 권유 등 본 약관 제7조에서 정한 차단 카테고리에
   해당하는 콘텐츠를 작성·전송하는 행위.
4. 타인의 음성·사진·이미지를 무단 사용하거나 보이스 클론을 사칭·도용 목적으로
   생성하는 행위.
5. 외부 채널 (메신저 ID, 전화번호, 이메일 등) 로 유도하는 스팸·광고 행위.
6. 서비스의 안전성을 저해하는 자동화된 접근, 크롤링, 역공학, 어뷰징 행위.

### 제7조 (콘텐츠 모더레이션)

본 조는 모든 메시지, 보이스 한마디, 프로필 사진에 적용되며, 카테고리 정의 및
차단 방식은 다음과 같습니다.

**(1) 차단 카테고리**

다음 4 개 카테고리에 해당하는 표현은 자동 차단됩니다.

- 성적 콘텐츠 (sexual)
- 마약 관련 콘텐츠 (drug)
- 미성년자 대상·관련 표현 (minor)
- 자해 권유의 타인 명령형 (self_harm — 한국 형법 §252, 일본 刑法 §202 자살관여죄
  형사처벌 정합)

(harassment / hate / violence 카테고리는 1차 출시 차단 대상에서 제외하며 운영
데이터 누적 후 결정합니다.)

(이용자 본인의 1인칭 위기 신호 — 예: "죽고 싶어" — 는 차단하지 않으며, helpline
연계 분기는 후속 출시 예정.)

**(1-1) 검증 절차 — 사전 키워드 차단 + OpenAI 2차 검수**

회사는 다음 2 단계로 콘텐츠를 검증합니다.

1. **사전 키워드 차단** — 회사가 운영하는 정규화된 키워드 사전 (한국어, 일본어,
   영어) 으로 메시지 및 보이스 한마디를 검사. 키워드 매칭 시 즉시 차단.
2. **OpenAI Moderation 2차 검수** — 사전 차단을 통과한 텍스트를 OpenAI Inc.
   (미국 캘리포니아 소재) 의 Moderation API (`omni-moderation-latest`) 로 송신해
   2차 검사. OpenAI 의 처리방침이 적용되며, 회사는 별도로 저장하지 않습니다.
   OpenAI 의 처리 정책은 https://openai.com/policies 참조.

차단 응답 본문은 카테고리·매칭 키워드를 노출하지 않습니다 (우회 패턴 학습 방지).

**(2) 자동 정지 (누적 신고)**

같은 회원에 대한 신고가 일정 수 (출시 시점 환경변수 `AUTO_FREEZE_REPORT_THRESHOLD`
기준 3 건) 누적되면 회사는 사전 통지 없이 해당 계정을 자동으로 정지할 수 있습니다.
정지된 회원은 앱 내 안내 모달을 통해 정지 사실 및 이의제기 채널 (CS 이메일) 을
확인할 수 있습니다.

**(3) audit 기록 보존 (1년)**

콘텐츠 차단 이력 (sender_id, 카테고리, 언어, 시각) 및 자동 정지 발동 이력
(frozen_user_id, 신고자 ID 배열, 발동 시각) 은 운영 목적 (재범자 식별, 신고 패턴
분석, 정지 처분의 근거 보존) 으로 **1 년간 보존** 후 자동 폐기됩니다. 메시지
원문 및 매칭 토큰은 audit 에 저장하지 않습니다.

### 제8조 (음성 데이터의 처리)

본 조는 회원이 등록한 보이스 클론 학습 음성, 메시지 음성, 보이스 한마디 음성 모두에
적용됩니다. 자세한 항목은 개인정보처리방침 제4조를 함께 참조하십시오.

**(1) 보이스 클론 학습 음성**

회원이 보이스 클론 생성을 위해 등록한 음성 샘플은 ElevenLabs 가 음성 ID 를
생성한 즉시 회사 인프라에서 폐기됩니다. 원본 학습 음성은 회사가 보관하지
않습니다.

**(2) ElevenLabs 음성 ID**

ElevenLabs 가 보유하는 회원의 음성 ID 는 회원이 보이스 클론을 재녹음할 때까지
유지되며, 재녹음 시 이전 음성 ID 는 ElevenLabs API 를 통해 삭제 요청됩니다.
회원 탈퇴 시에도 회사가 ElevenLabs API 로 삭제를 요청합니다.

회원은 보이스 클론 단독 삭제를 별도로 요청할 수 없으며, 재녹음(덮어쓰기) 또는
계정 탈퇴를 통해서만 삭제할 수 있습니다.

**(3) 메시지 음성 (30 일 자동 폐기)**

채팅에서 합성된 메시지 음성 파일은 **수신자가 청취한 시점으로부터 30 일이 경과**
하면 회사의 Storage 에서 자동으로 폐기됩니다. 텍스트 및 자동 번역 결과는 매치
생명주기와 묶여 유지됩니다. 회원이 폐기된 음성을 다시 듣고 싶을 때는 ElevenLabs
재합성을 요청할 수 있으며, 이 경우 회원의 현재 보이스 클론으로 새로 합성됩니다
(재녹음으로 음성 ID 가 변경된 경우 새 음성이 합성될 수 있습니다).

미청취 메시지 음성은 30 일 경과 이후에도 자동 폐기 대상이 아닙니다 (수신자가
청취할 때까지 보존).

**(4) 보이스 한마디 음성**

보이스 한마디 음성은 회원이 텍스트를 변경하거나 계정을 탈퇴할 때 폐기됩니다.

### 제9조 (사진 데이터의 처리)

본 조는 회원이 등록한 프로필 사진 (원본 및 AI 변환본) 에 적용됩니다. 자세한
항목은 개인정보처리방침 제5조를 함께 참조하십시오.

**(1) 원본 사진 — 변환 후 즉시 폐기**

회원이 업로드한 원본 사진은 OpenAI 의 이미지 생성 모델 (gpt-image-2) 로 수채화·
일러스트 풍 변환본을 생성한 직후 회사 인프라에서 폐기됩니다. 원본 사진은 회사가
보관하지 않으며, 회원에게 다시 다운로드하거나 노출되지 않습니다.

**(2) AI 변환본 — 프로필 사진으로 사용**

회사는 변환본을 회원의 프로필 사진으로 매칭 탐색 및 매치 화면에서 노출합니다.
변환본은 회원 탈퇴 시까지 회사 Storage 에 보관되며, 탈퇴 시 일괄 삭제됩니다.

AI 변환은 외관을 일러스트 톤으로 stylize 할 뿐 실제 나이·외모·신원을 변경하지
않습니다. 본 서비스의 모든 회원은 가입 시 만 18 세 이상임을 자기신고하였으며,
사진의 외관과 무관하게 미성년자 사칭이 적발될 경우 영구 차단 조치됩니다.

**(3) 매치 진행에 따른 원본 공개**

매칭 후 회원 간 채팅이 10 회 왕복에 도달하면 회원의 원본 등록 사진 전체 (최대
5 장) 가 상대에게 공개됩니다. 이는 회원이 가입 시점에 동의한 매칭 정책의 일부
이며, 회원은 사진 업로드 시 이 공개 정책에 동의합니다.

**(4) 모더레이션 거부**

OpenAI 의 safety filter 가 사진 변환을 거부할 경우 원본 및 변환본 모두 저장
되지 않으며, 회원에게 재업로드를 안내합니다. 거부 사유는 차단 카테고리 표면
'photo' 로 audit 에 기록되며 1 년 후 자동 폐기됩니다 (제7조 모더레이션 audit
정책 정합).

### 제10조 (회원 탈퇴 및 데이터 삭제)

1. 회원은 언제든지 서비스 내에서 탈퇴를 요청할 수 있습니다.
2. 탈퇴 시 회원의 프로필, 매치, 메시지, 사진 변환본, 차단·신고 audit, 자동 정지
   audit, 디바이스 푸시 토큰, ElevenLabs 음성 ID (외부 삭제 요청 포함) 는 모두
   삭제 또는 익명화됩니다.
3. 단, 관련 법령상 보존 의무가 있는 정보는 해당 기간 동안 별도 분리 보관 후
   파기합니다.

### 제11조 (책임의 제한)

1. 회사는 천재지변, 외부 서비스 (ElevenLabs, OpenAI, Vertex AI, Supabase) 의
   장애, 회원의 귀책 사유로 인한 손해에 대해 책임을 지지 않습니다.
2. 회사는 회원 간의 분쟁에 직접 개입하지 않으며, 다만 콘텐츠 모더레이션 정책
   (제7조) 에 따라 차단·정지 조치를 취할 수 있습니다.

### 제12조 (준거법 및 관할)

본 약관은 대한민국 법에 따라 해석되며, 분쟁 시 회사 소재지 관할 법원을
1심 관할 법원으로 합니다.

## Part 2. 개인정보처리방침

### 제1조 (수집 항목 및 목적)

회사는 다음 정보를 수집·이용합니다.

| 항목 | 수집 목적 | 보존 기간 |
|------|----------|----------|
| 이메일, 비밀번호 (또는 Google OAuth 식별자) | 회원 식별, 인증 | 회원 탈퇴 시까지 |
| 생년월일 | 만 18 세 이상 확인, 매칭 연령 필터 | 회원 탈퇴 시까지 |
| 성별, 국적, 사용 언어 | 매칭 알고리즘 (성별·언어·국적 선호도) | 회원 탈퇴 시까지 |
| 프로필 사진 원본 | AI 변환본 생성 입력 (gpt-image-2) | **변환 후 즉시 폐기** |
| 프로필 사진 AI 변환본 | 매칭 탐색 노출, 채팅 상대 식별 | 회원 탈퇴 시까지 |
| 보이스 클론 학습 음성 | ElevenLabs 음성 ID 생성 | **생성 즉시 폐기** |
| ElevenLabs 음성 ID | 메시지 음성 및 보이스 한마디 합성 | 재녹음 시 / 탈퇴 시 외부 삭제 요청 |
| 보이스 한마디 텍스트 | 매칭 탐색 시 자기소개 노출 | 회원 탈퇴 시까지 |
| 보이스 한마디 음성 (ko/ja/en 3 슬롯) | 매칭 탐색 시 청취 | 텍스트 변경 또는 탈퇴 시 폐기 |
| 메시지 텍스트 및 번역 | 채팅 기능 | 매치 종료 또는 탈퇴 시까지 |
| 메시지 음성 파일 | 채팅 음성 재생 | **청취 + 30 일 자동 폐기** |
| 매칭 선호도 (나이, 성별, 언어, 국가) | 매칭 알고리즘 | 회원 탈퇴 시까지 |
| 디바이스 푸시 토큰 | 메시지 / 매치 알림 발송 | 토큰 등록 해제 또는 탈퇴 시 |
| 콘텐츠 차단 이력 (sender_id, 카테고리, 언어, 표면 — message/voice_intro/photo) | 운영, 재범자 식별 | **1 년** 후 자동 폐기 |
| 자동 정지 이력 (frozen_user_id, 신고자 ID 배열, 발동 시각) | 운영, 정지 처분 근거 보존 | **1 년** 후 자동 폐기 |

### 제4조 (음성 데이터의 처리)

본 조는 약관 제8조의 정책을 정보 처리 관점에서 보완합니다.

**(1) 학습 음성 — 즉시 폐기**

보이스 클론 생성을 위해 회원이 등록한 음성 샘플은 ElevenLabs API 로 음성 ID 가
생성된 직후 회사 인프라에서 폐기됩니다 (저장 컬럼 및 Storage 버킷 모두 부재).
원본 학습 음성은 회사가 보관하지 않으며, 재청취 기능도 제공하지 않습니다.

**(2) ElevenLabs 음성 ID — 외부 잔존**

ElevenLabs 가 보유하는 회원의 음성 ID 는 회원의 메시지 및 보이스 한마디 합성에
사용됩니다. 회원이 보이스 클론을 재녹음할 때 회사는 이전 음성 ID 의 ElevenLabs
API 삭제 요청을 발송하며, 회원 탈퇴 시에도 동일한 삭제 요청을 발송합니다. 삭제
요청의 최종 처리는 ElevenLabs 의 정책 및 처리 시한에 따릅니다.

회원은 보이스 클론을 단독 삭제할 수 없으며, 재녹음 (덮어쓰기) 또는 계정 탈퇴
경로로만 삭제할 수 있습니다. 이는 데이팅 서비스의 특성상 보이스 클론 임의 삭제로
인한 다른 회원의 매치 음성 재생 회귀 (영구 청취 불가) 를 방지하기 위함입니다.

**(3) 메시지 음성 — 30 일 자동 폐기 + 재합성**

채팅에서 합성된 메시지 음성 파일 (`voice-messages` 버킷) 은 **수신자가 청취한
시점으로부터 30 일이 경과** 하면 회사의 정기 sweep 작업으로 자동 폐기됩니다.
메시지 텍스트 및 번역 결과는 매치 생명주기와 묶여 유지되며 음성 파일만 별도
폐기됩니다.

회원이 폐기된 음성을 다시 듣고자 할 때는 회사의 재합성 라우트를 통해 ElevenLabs
로 새 음성 파일이 합성됩니다. 재합성된 음성은 회원의 현재 보이스 클론으로 합성
되므로, 재녹음으로 음성 ID 가 변경된 경우 첫 발신 시점과 다른 음성이 합성될 수
있습니다 (의도된 정책).

미청취 메시지 음성은 30 일 경과 이후에도 자동 폐기되지 않습니다. 수신자가 청취
완료한 시점부터 30 일 카운트가 시작됩니다.

**(4) 보이스 한마디 음성 — 텍스트 변경/탈퇴 시 폐기**

보이스 한마디는 ko/ja/en 3 슬롯으로 합성되어 `voice-intro-audio` 버킷에 저장
됩니다. 회원이 보이스 한마디 텍스트를 변경하면 이전 음성 파일은 자동 폐기되며
새 텍스트로 재합성됩니다.

### 제5조 (사진 데이터의 처리)

본 조는 약관 제9조의 정책을 정보 처리 관점에서 보완합니다.

**(1) 원본 사진 — 변환 후 즉시 폐기**

회원이 업로드한 원본 사진의 raw bytes 는 OpenAI gpt-image-2 API 로 변환본이 생성
되는 즉시 회사 인프라에서 폐기됩니다 (Storage 임시 경로 삭제, DB 컬럼 미보관).
원본은 회사가 보관하지 않으며, 다시 다운로드하거나 노출되지 않습니다.

**(2) 변환본 저장 위치 및 보존**

회사는 변환본을 Supabase Storage 의 `photos` 버킷에 저장하며, 회원 탈퇴 시
`auth.cleanupDeletedUserAssets` cleanup task 로 일괄 삭제합니다.

**(3) OpenAI 학습 미활용 + 변환본의 비-상업적 사용**

원본 사진의 raw bytes 는 OpenAI 의 API Data Usage Policy
(https://openai.com/policies/api-data-usage-policies) 에 따라 OpenAI 의 모델 학습에
사용되지 않습니다 (2023-03-01 이후 API 입력은 학습 미활용 정책). 변환본은
회원 본인의 프로필 사진으로만 사용되며 회사는 별도의 상업적 활용을 하지
않습니다.

**(4) 사진 모더레이션 — surface='photo'**

원본 사진은 변환 단계에서 OpenAI 의 safety filter 검사를 거치며, 거부 시 차단
카테고리 표면 'photo' 로 `moderation_blocks` audit 에 기록됩니다. 이 audit 의 보존
기간은 제1조 표의 콘텐츠 차단 이력 항목에 통합됩니다 (1 년).
