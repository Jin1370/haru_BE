# DPA 트래커 (LAUNCH_CHECKLIST #11)

수탁사 처리계약(DPA). 개인 개발자가 **직접 협상할 게 아니라**, 벤더가 만들어둔
표준 DPA를 **자동 적용 / 한 번 수락 / 메일 한 통**으로 갖추면 끝. 소급이 안 되니
미룰 이유 없음(전부 무료, ~1시간).

법적 근거: PIPA §26·§28-8 / GDPR Art.28·Art.46. #4(방침 수탁사 명시)·#5(동의)는
이미 완료 — DPA는 그 위에 "계약" 한 겹.
